Title: 被掏空的LL
Time Limit: 1 Sec
Memory Limit: 32 MB

Description
经过双十一的奋战，LL已经掏空了自己的身体，拿不出钱来吃饭了。然而土财主金子大爷腰缠万贯，决定在接下来的$N$天施舍LL，每天给LL若干元钱。
但金子大爷太有钱了，对钱的数量已经没有感觉了，以至于不知道每天给了LL多少钱。
但可以肯定的是，这$N$天，金子大爷给了LL一共$M$元。LL一天需要$K$元才能在那天吃饱饭，而且她十分贪吃，不会存钱，当天获得的钱一定全部用完（即使吃不饱也会花完）。
在这$N$天中，如果LL一天饱饭都没吃到，她就会饿死。换句话说，LL只要在这$N$天中有一天吃饱了饭，她就不会饿死。
问题来了，过完这$N$天，她可能会饿死吗？


Input
输入的第一行为一个整数$T$，代表有$T$ ($T$实际输入约为1000) 组数据
每组数据为一行，三个整数$N$, $M$, $K$ （0 < $N$ < $10^9$, 0 <= $M$, $K$ < $10^9$）

Output
对于每组样例, 输出一行 “Case #$x$: $y$”, $x$代表组号，从1开始, $y$代表结果，若LL可能饿死，结果是Yes，否则结果是No.

Sample Input
2
1 2 2
2 2 2

Sample Output
Case #1: No
Case #2: Yes

HINT
此题货币的最小单位为元。
假如LL在某天被施舍了x元，若x >= k，那一天她就吃饱了饭，N天后不会饿死。特别地，如果k等于0，说明她每天不花钱也能吃饱饭。
对于第一组输入，LL要接受大爷1天的施舍，大爷总共会给2元，因为只有一天，所以LL在这一天一定会获得2元，2 >= 2，吃饱了饭，不可能饿死。
对于第二组输入，LL要接受大爷2天的施舍，大爷总共会给2元，有可能大爷第一天给她1元，第2天给1元，1 < 2，LL两天都没吃饱饭，于是就饿死了。
