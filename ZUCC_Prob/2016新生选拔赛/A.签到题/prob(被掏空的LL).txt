Title: 复读机
Time Limit: 1 Sec
Memory Limit: 32 MB

Description
你群总是有那么一群复读机，当他们看到你的名字的时候，就会脱口而出：“你的名字 dalao!”。
现在你发送了一个名字，并且知道群里有$N$个复读机，每个复读机会念一遍“你的名字 dalao!”
请你输出群里的聊天记录，具体见样例

Input
输入的第一行为一个整数T，代表有 $T$ ($T$ <= 10) 组数据
每组数据一行，有一个字符串$str$（题目保证其长度不超过80，且只由小写字母组成），一个整数$N$(1 <= $N$ <= 10)
字符串和整数用一个空格隔开。

Output
对于每一组数据，输出$N$行复读机说的话。

Sample Input
3
ll 3
zx 1
jzst 2

Sample Output
ll dalao!
ll dalao!
ll dalao!
zx dalao!
jzst dalao!
jzst dalao!

HINT
